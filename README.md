# SIGAP DESA — App

Sistem Informasi Gerak Aktif Pelayanan Desa — Desa Bojongkulur.

## Struktur

Root folder ini **adalah** aplikasi Next.js (tanpa subfolder `frontend/`).

```
sigap-desa-app/
├── src/app/          # Routes (App Router)
├── src/components/
├── src/lib/
└── public/
```

## Menjalankan

```bash
npm install
npm run dev
```

Buka http://localhost:3000
